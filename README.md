# jacoco-on-gradle-sample

### Run tests and generate JaCoCo report
- `test` task를 실행하면 `jacocoTestReport` task가 따라서 실행

```bash
$ ./gradlew clean test
$ ./gradlew clean test jacocoTestReport jacocoTestCoverageVerification
```
